# Сервис Mesto

Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

**В проекте используются:**
* HTML5, CSS3
* JavaScript
* БЭМ
* Организация файловой структуры Nested
* Адаптивная верстка (Desktop First)
* CSS Grid Layout
* CSS Flexbox Layout
